Array v/s Object in JavaScript

Array

In simple terms , array can be defined as a collection of elements or list of items.

Like : A collection of numbers [1,2,3,4] , collection of cars ["Alto" , "Audi" , "Mahindra"] etc.

Arrays in JavaScript comes with bundle of methods and properties such as:

  • length
  • push
  • pop
  • shift
  • unshift
  • concat
  • includes
  • indexOf
  • reverse
  • sort
and many more.....

Let us discuss in detail about the above methods and properties :

  1. length : Like string , array in JavaScript have length property
    For example
    var colors = ['red' , 'orange' , 'green']
    console.log(colors.length) will give us 3 because there are three elements in colors array


  2. push : push is a method in JavaScript which is used to insert new value at it's end
    For example
    var colors = ['red' , 'orange' , 'green']
    colors.push('pink') will make colors array ['red' , 'orange' , 'green' , 'pink']


  3. pop :pop is a method which is used to remove value from the end of the array
    For example
    var colors = ['red' , 'orange' , 'green' , 'pink']
    colors.pop() will make colors array ['red' , 'orange' , 'green']


  4. shift :This method is used to remove value from the front of the array
    For example
    var colors = ['red' , 'orange' , 'green' , 'pink']
    colors.shift() will make colors array ['orange' , 'green' , 'pink']


  5. unshift :This method is used to insert new value at the front of the array
    For example
    var colors = ['red' , 'orange' , 'green' , 'pink']
    colors.unshift('yellow') will make colors array ['yellow' , 'red' , 'orange' , 'green' , 'pink']


  6. concat :concat is a method which is used to join arrays
    For example
    var cats = ['blue' , 'witty']
    var dogs = ['rusty' , 'wyatt']
    cats.concat(dogs) will give us ['blue' , 'witty' , 'rusty' , 'wyatt'] . However , the original arrays i.e cats and dogs array will remain unchanged


  7. includes :This method is used to check whether a given array contains a particular value or not
    For example
    var colors = ['red' , 'orange' , 'green' , 'pink']
    colors.includes('red') will return true because colors array contains 'red'


  8. indexOf :This method returns the index of first occurrence of particular value in the array
    For example
    var colors = ['red' , 'orange' , 'green' , 'pink']
    colors.indexOf('green') will return 2 , the important thing to note here is array in JavaScript has 0-based indexing


  9. reverse :This method is used to reverse the array
    For example
    var colors = ['red' , 'orange' , 'green' , 'pink']
    colors.reverse() will make colors array ['pink' , 'green' , 'orange' , 'red']


  10. sort :sort method is used to sort the array , it is not necessary that the array will be sorted in ascending or descending order because sorting in JavaScript is done on the basis of UTF codes.




Object

JavaScript objects are collection of properties , consider it as a real world entity

Properties are key-value pairs , if you are from Python background then you can relate objects to dictionary in python


eg: var fitBitData = {
totalSteps:30,
totalMiles:27
};

Accessing object properties in JavaScript: The properties of an object can be accessed in either of the following ways:

  1. fitBitData.totalMiles
  2. fitBitData[ "totalMiles" ]


Here are some of the important things that you should know about objects in JavaScript:
  • Internally, all the keys are stored as a string , so you can define key of an object as a number also apart from string.

  • We can also change the value associated with a key , e.g : fitBitData.totalSteps=100

  • Object in JavaScript , unlike arrays and string is not iterable.

  • Object.keys(objectName) returns array of all keys of objectName

  • Object.values(objectName) returns array of all values of objectName
